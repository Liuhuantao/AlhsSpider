# AlhsSpider

艾利浩斯图书馆小说爬虫，将 https://alhs.xyz 的文章批量下载为本地 txt 文件。

## 特性

- **全站爬取** — 自动遍历所有列表页，收集文章链接
- **智能过滤** — 跳过字数 < 500 的文章、分类为「互动小说」的文章
- **章节合并** — 自动检测多章节系列，合并为单个 txt 文件
- **断点续传** — 进度保存到 `progress.json`，中断后重跑不重复下载
- **页码范围** — 支持指定起始/结束页数
- **多格式兼容** — 自动适配标准 `<p>`、飞书 `<div.ace-line">`、旧版裸 `<div>` 三种段落格式

## 依赖

- Python 3.10+
- requests
- beautifulsoup4
- lxml

```bash
pip install requests beautifulsoup4 lxml
```

## 用法

```bash
# 全量爬取（从第 1 页到最后一页）
python main.py

# 指定页码范围
python main.py --start-page=5 --end-page=20

# 从某页开始，直到最后
python main.py --start-page=50

# 清除进度重新开始
python main.py --reset

# 测试：只扫描 5 页，下载 10 篇
python main.py --max-pages=5 --max-articles=10

# 只收集链接不下载（查看有多少文章）
python main.py --collect-only
```

| 参数 | 说明 | 默认值 |
|---|---|---|
| `--start-page` | 起始列表页 | 1 |
| `--end-page` | 结束列表页 | 0（直至最后） |
| `--max-pages` | 最多扫描页数（等同于 `--end-page`） | 0 |
| `--max-articles` | 最多下载篇数（测试用） | 0 |
| `--reset` | 清除进度从头开始 | — |
| `--collect-only` | 只收集链接不下载 | — |

## 输出

文章保存在 `novels/` 目录，每篇一个 txt 文件：

```
novels/
├── 《地衣草》.txt
├── 迷失.txt              ← 5 章合并
├── 器灵小姐讨厌恋物癖.txt
└── ...
```

文件格式：

```
标题：《地衣草》
作者：Vapa
日期：2026-4-22 8:10
分类：原创小说
字数：33793
原文：https://alhs.xyz/index.php/archives/2026/04/73762/

============================================================

（正文内容）
```

多章节合并文件：

```
标题：迷失
============================================================

第1章 迷失 第一章
----------------------------------------
（正文）

第2章 迷失 第二章
----------------------------------------
（正文）
...
```

## 过滤规则

文章在下载阶段被过滤（不是收集阶段），满足以下任一条件则跳过：

- 分类包含「互动小说」
- 标注字数 < 500
- 未标注字数且实际正文 < 500 字符

章节合并后的系列以**总字数**判断是否过滤。

## 进度文件

`progress.json` 记录已完成 URL、已合并系列、收集的链接等。删除此文件或使用 `--reset` 可重新开始。
